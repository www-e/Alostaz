## Getting Started

### Prerequisites
- Node.js 16+
- npm 8+
- PostgreSQL 13+
- Supabase CLI (for local development)

### Local Development

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables (see `.env.example`)
4. Run the development server: `npm run dev`
